# History And Iterations

Aqui se guardaran los analisis detallados cerrados por entrega.

Regla sugerida:

- un archivo por HU cuando el cambio es acotado
- un archivo por iteracion cuando la entrega agrupa varias HUs

Cada archivo debe basarse en la plantilla de `../templates/post-implementation-analysis-template.md`.

Baseline disponible:

- `current-state-up-to-HU-008/`: analisis consolidado del estado actual de la solucion hasta HU-008.
- `current-state-up-to-HU-009/`: analisis consolidado del estado actual de la solucion hasta HU-009, incluyendo gestion multiempresa a nivel cuenta.
- `current-state-up-to-HU-010/`: analisis consolidado del estado actual de la solucion hasta HU-010, incluyendo Locations y Work Centers tenant-scoped.
